# Design

## Goal

The goal of this exercise is to teach students about optional values enabled by the `Option` type and what you can do with them.

## Learning objectives

- Know of the existence of the `Option` type.
- Know how to create an `Option` value.
- Know how to pattern match on its two values.
- Know how it relates to optional values.
- Know how to return a default value.

## Out of scope

- The generic concept of pattern matching; this exercise focuses on `Option` patterns.
- The higher order `Option` functions such as `map`, `iter`, etc.

## Concepts

- `options`

## Prerequisites

- `records`
- `tuples`
